Música

La instrucció més important per fer música amb arduino és "tone" i té 3 arguments (tercer optatiu):

  • El primer argument és "El pin", la posició de Arduino en la qual connecto: pin 8 o pin 4.
  • El segon argument és "La freqüència" i s'expresa en Herz (Hz)
  • El tercer argument és optatiu i és "el temps de durada"
void setup() { tone(4, 500); } void loop() {}
